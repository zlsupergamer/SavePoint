## Descrição
Sistema desenvolvido em C# para catalogar jogos, registrar conquistas e 
acompanhar estatísticas de progresso da biblioteca pessoal do jogador.

## Pré-requisitos

- .NET SDK 10.0 ou superior
- Download: https://dotnet.microsoft.com/download

## Como executar

1. Baixe o repositório (botão verde **Code** → **Download ZIP**) e extraia a pasta
2. Abra o terminal dentro da pasta extraída
3. Execute o comando:

dotnet run --project SavePoint.csproj
